1¿Qué es Next.js y por qué se dice que es el framework de React más popular? Es un freamwork que permite crear un servidor renderizado, con sitios estáticos y un manejo de rutas automática, popular debido al la facilidad de desarrollo con características de routing, apis routes y optimización 
2 ¿Cuál es el comando exacto con npx para inicializar un proyecto de Next.js asegurando usar la última versión? npx create-next-app@latest Este comando ejecuta e instala ejecuta automática una aplicación de next desde 0 
3 ¿Cuál es el comando utilizado para levantar el entorno de desarrollo y en qué puerto se ejecuta por defecto? npm run dev 
4 ¿Qué función cumple el comando npm run build en un proyecto de Next.js? Crae una versión optimizada del proyecto, asegurando que los archivos y recursos estén listos para ser desplegados en un servidor 
5 Describe para que sirven las carpetas, public y app en la carpeta public, estarán los archivos estáticos, como imágenes, videos donde se pueden acceder desde el navegador , por otro lado, en la carpeta app, es la estructura de la aplicación, donde se definen rutas por anidacion de carpetas y archivos 
6 ¿Que es un CCS MODULE y como se usan? Un css modulo, es un archivo css con alcance local, donde pudes usarlo en tantas veces como quieras, en 8 componentes diferentes, evita conflictos globales 
7 ¿Que hace el componente Layout.tsx ? define la estructura compartida de la aplicación, como encabezados, navegación, pie de página, envuelve las los archivos como page.tsx 
8 ¿Que es el enrutamiento ? Es el sistema que permite navegar entre diferentes paginas de la aplicación, mediante rutas, que son generadas automáticamente a partir de la estructura de carpetas 
9 ¿Cual es la diferencia entre Image e img? Que img mantiene un formato que no esta optimizado y es más pesado, Image lo optimiza, pesando menos pero cambia el formato según el formato idóneo para el navegador, dependiendo como se manipule el navegador en pc o en movile cargara el tipo de imagen mas optimo 
10 ¿Que es el Lazy Load? Es una técnica que consiste en cargar recursos solo caundo son requeridos, en vez de cargarlos todo

12 ¿Como funciona el enrutamiento en Next JS ? Funciona mediante un sistema basado en archivos,donde se estructura a partir de carpetas, dentro de la capeta app, cada carpeta es un segmento de la url 
13 ¿Que es una ruta en el contexto web y donde las hemos visto en la clase ? Es una url que permite el accesos de una sección en especifica 
14 ¿Que arrojo a MiduDev el localhost:300/dashboard, donde hemos visto esos errores en la clase ? Un error 404, indicando que la página no fue encontrada, se vio al hacer un get en el servidor que no tiene 
15 ¿Que es el File System Routing? Un sistema de enrutamiento donde las rutas de la aplicación se generan automáticamente por archivos y carpetas 
16 ¿Como funcionan las rutas anidadas? Mediante la creación de carpetas dentro de otras carpetas 
17 ¿Investiga por que es necesario esto ? Permite exportar componentes como elprincipal del archivo, con el fin de ser importado en otros archivos, en page.tsx y layout.tsx es obligatorio para que funcionen como rutas 
18 ¿Para que sirve el componente Link? Para que haya un navegación fluida entre páginas, reutilizando los recursos posibles y renderice solo lo que cambia 
19 ¿Que se carga cuando usamos <a> y que cuando se usa <link> ? Cuando usa <a> todos los recursos , pero al momento de cambiarlo a <Link> carga solo lo necesario, cambiando el título, la url y el contenido, y se nota mas fluido, por cargar lo necesario
